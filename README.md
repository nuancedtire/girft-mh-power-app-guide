# GIRFT MH ED Breach Analysis Power Apps Guide

Static GitHub Pages implementation guide for a non-identifiable Power Apps and SharePoint workflow.

This repository does not contain patient data, SharePoint credentials, or a deployable Power App. The Power App must be created and published within an approved NHS Microsoft 365 environment.

## Local preview

Open `index.html` in a browser, or run:

```bash
python -m http.server 8080
```

Then visit `http://localhost:8080`.
